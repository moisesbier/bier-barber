# Bier Barber

Olá, bem-vindo ao meu projeto onde desenvolvi uma página de barbearia moderna, com muito estilo e animações ao rolar da página.

## Ferramentas utilizadas:

* HTML

* CSS

* SASS

* AOS Animate

* Keyframes

* Font Awesome

* API do WhatsApp
